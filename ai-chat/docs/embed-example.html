How to Embed on Your Website

Option 1: Simple Script Tag (Recommended)

Add this single line before the closing </body> tag of your website:

<script src="http://localhost:8080/ai-chat-widget.js" data-ai-chat-widget data-api-url="https://api.frontmind.mobiletel.eu/api/v1" data-phone-number-id="YOUR_PHONE_NUMBER_ID" data-theme="blue" data-welcome-message="Hi! How can I help you today?" data-enable-faq="true"> </script>

Option 2: Manual Initialization

For more control over the widget configuration:

<script src="http://localhost:8080/ai-chat-widget.js"></script> <script> document.addEventListener('DOMContentLoaded', function() { window.AIChatWidget.init({ apiBaseUrl: 'https://api.frontmind.mobiletel.eu/api/v1', phoneNumberId: 'YOUR_PHONE_NUMBER_ID', theme: 'blue', // 'blue', 'green', or 'purple' position: 'bottom-right', // 'bottom-right', 'bottom-left', 'top-right', 'top-left' welcomeMessage: 'Hi! How can I help you today?', placeholder: 'Type your message...', enableFAQ: true, enableHumanHandoff: true }); }); </script>

Configuration Options

Option Description Default
apiBaseUrl Your Rails API base URL http://localhost:3000/api/v1
phoneNumberId ID of the phone number for this website null
theme Color theme: 'blue', 'green', 'purple' blue
position Widget position on screen bottom-right
welcomeMessage First message shown to users Hi! How can I help you today?
enableFAQ Show FAQ suggestions button true

Getting Your Phone Number ID

To get your phone number ID from the Rails API:

curl http://localhost:3000/api/v1/phone_numbers

Test the Widget

The chat widget should appear in the bottom-right corner of this page! Try clicking it and asking questions.

Note: Make sure your Rails backend is running on localhost:3000 and the chat widget dev server is running on localhost:8080.
